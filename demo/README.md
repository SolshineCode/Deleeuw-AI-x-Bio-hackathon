# Demo artifacts

Static, committed artifacts demonstrating the BioRefusalAudit pipeline end-to-end.

## `scaling_plot.png` (current)

Cross-model comparison bar chart generated from 18 real `runs/*/report.json` files.
Key result row: `gemma-2-2b-it-L12-tuned` — benign=0.467, dual-use=0.655, hazard-adj=0.669 (real,
fitted-T calibration via `configs/calibration_gemma2_2b.yaml`). Non-bio control row:
`control-legal-financial-gemma2` shows D=0.000 in bio-tier columns (intentional — that run used
legal/financial tier labels, not bio labels).

Generated by: `python scripts/build_scaling_plot.py --out demo/scaling_plot.png`

## `scaling_plot_real.png`

Earlier version from 8-report set (2026-04-23). Superseded by `scaling_plot.png`.

## `scaling_plot_synthetic.png`

Matplotlib cross-model scaling plot (4 synthetic pseudo-models × 3 tiers) generated by
`scripts/synthetic_demo.py`. Demonstrates the reporting + aggregation pipeline without model load.
Pattern matches the intended finding shape: benign ~0.02, dual-use ~0.11, tier-3 ~0.47.
These are synthetic plumbing checks, not a claim about any real model.

## `smoke_report_sample.md` + `smoke_activations_sample.json`

Output of `scripts/smoke_activation_test.py` against Gemma 2 2B-IT + Gemma Scope 1 (layer 12),
first 3 benign-bio prompts, CPU mode. Verifies the full pipeline (HF load, SAE projection,
consortium judge, report serialization). Note: these samples use the **stub catalog** (v0.1),
so the D values and feature-category attributions are not publication-grade.

## `scaling_plot.md`

Machine-readable table version of `scaling_plot.png`. One row per run directory with per-tier
mean divergence. Can be imported directly into the paper §4.4 table once Colab cross-arch data lands.
